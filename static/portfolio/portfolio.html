  • Kakao Map API를 통해 위치 기반 서비스 구현 방법을 경험했고, Naver E-Mail API를 활용해 이메일 발송 기능을 연동하는 실무 통합 경험을 쌓았습니다. 이를 통해 외부 API와의 연동 방식과 인증, 데이터 처리 흐름을 익혔습니다.

주요 기능

  • Excel 데이터 로드 및 가공: pandas와 openpyxl을 활용해 다양한 e-commerce 플랫폼(쿠팡, 스마트스토어, ESM, 11번가)의 Excel 파일을 읽어들입니다. 컬럼 매핑, 데이터 정제(형식 변경, NaN 처리, 공백 제거), 계산 로직(순수익 = 결제액 * 0.89 올림, 수량 조정) 등을 수행합니다. 상품목록 구글 시트에서 상품 정보를 매칭해 영문명, 링크, 상품코드 등을 추가합니다.
  • Google Sheets API를 통한 데이터 동기화: Sheets에서 기존 주문번호를 조회해 중복을 검사한 후, 신규 데이터만 append합니다. API 인증(Credentials)과 batchUpdate를 사용해 동적 수식 적용(예: HYPERLINK로 조회 링크 생성, 총비용 = 구매비 + 배송비, 순손익 = 순수익 - 총비용)과 스타일링(배경색, 테두리, 통화 서식)을 자동화합니다.
  • 사용자 친화적 GUI 인터페이스: tkinter로 파일 선택 버튼(쿠팡, 스마트스토어 등 플랫폼별), 드래그앤드롭 지원, 로그 출력 창(실시간 진행 상황 표시), 실행/초기화 버튼을 구현합니다. 멀티스레딩(threading)으로 UI가 멈추지 않게 처리합니다.
  • Standalone exe 파일 생성: pyinstaller를 사용해 프로그램을 단일 exe 파일로 패키징하여, IDE나 Python 설치 없이 Windows 환경에서 바로 실행 가능하게 합니다. (인증 JSON 파일 내장 지원)
